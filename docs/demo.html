Easily add a book discovery widget to any website! Explore top-rated books by genre with cover images, filtering, and personalized recommendations.

Get personalized book suggestions

📱 Responsive

Works on desktop and mobile

🚀 Quick Embed (Recommended)

Just add these two lines to your HTML:

<!-- Load the embed script --> <script src="https://TayyebaSadaq.github.io/goodreads-scraper/embed.js"></script> <!-- Create widget container --> <div id="book-widget" data-book-explorer data-width="100%" data-height="800px"></div>

⚙️ Custom Embed

For more control over initialization:

<script src="https://TayyebaSadaq.github.io/goodreads-scraper/embed.js"></script> <div id="my-book-explorer"></div> <script> BookExplorerWidget.create('my-book-explorer', { width: '100%', height: '600px' }); </script>

🖼️ Simple iframe Embed

Basic iframe embed (less features but universal compatibility):

<iframe src="https://TayyebaSadaq.github.io/goodreads-scraper/" width="100%" height="800px" frameborder="0" style="border-radius: 12px; max-width: 1200px;"> </iframe>

📝 Customization Options

Option Default Description
data-width 100% Widget width (e.g., "500px", "100%")
data-height 800px Widget height (e.g., "600px", "100vh")
